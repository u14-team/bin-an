# BIN-AN
``` Автоматический бот для торговли на Binance ```

BIN-AN - это бот, который торгует за Вас на криптобирже Binance. Благодаря ему Вы можете заниматься своими делами пока автоматика приносит Вам прибыль.
## Особенности
- Легкий JSON конфиг
- Красивый консольный интерфейс с подробной информацией
- Уведомления в Telegram
- Работа с несколькими криптовалютами одновременно
- Можно начинать с любым балансом
- Гибкая настройка

> Бот оценивает текущее состояние валюты и покупает её только в том случае, если она начала подниматься после долгого падения, при этом бот не закупит валюту, если её цена выше средней (`можно настраивать`).
> После покупки бот ожидает рост валюты на определённый процент, указанный в конфиге для валюты с учётом комиссии. Как только цена продажи будет достаточно выгодной, бот продаст валюту и вернёт оповещение.

`Разработчики бота не несут ответственности за Ваши. Использование бота не гарантирует 100% заработок, также, как и любые другие методы торговли на биржах.`

## Запланированные функции
- Стратегия HODL - бот будет держать валюту выше процента продажи пока она не начнёт падать.
- Усреднение - бот будет докупать валюту, если она упадёт на определённый процент после покупки.
- Бот в ТГ с подробной статистикой.e backend

## Установка и настройка
Для начала Вам нужен аккаунт в Binance, но он обязательно должен быть зарегистрирован по нашей реферальной ссылке https://www.binance.cc/ru/register?ref=LDKZLB0K или с использованием реферального кода `LDKZLB0K`

Теперь скачиваем бота и просто запускаем его. Бот создаст пустой конфиг и выключится, Вам нужно открыть config.json и настроить его согласно следующей таблице:
| Поле | Тип | Описание |
|-|-|-|
| binance | object | [Данные для авторизации в Binance](https://www.binance.com/ru/support/faq/360002502072) |
| binance.apiKey | string | Ключ API |
| binance.apiSecret | string | Секрет API |
| mainCurrency | string | Валюта для торга. (`USDT`) |
| currencies | object[] | Список валют, которыми бот будет торговать. |
| currencies[].slug | string | Имя валюты (`DOGE`) |
| currencies[].minProfit | float | Минимальный профит для продажи в дробной части. 1 - 100%, 0 - 0%. Если выставить более высокое значение, то бот будет держать валюту дольше, соответственно выше и риск. Профит считается с учетом комиссии, то есть если указать 0, то бот будет торговать в 0. (`0.005`) |
| currencies[].maxBuy | float | Максимальная сумма, которую можно потратить на эту валюту |
| currencies[].minBuy | float | Минимальная сумма, которую можно потратить на эту валюту |
| currencies[].minRun | float | Минимальная сумма валюты, при которой она будет считаться активной (то есть бот будет продавать её, а не покупать) |
| currencies[].flags | string[] | Дополнительные флаги для валюты, в будущем будут обозначать доп. стратегии. |
| currencies[].avgLimit | float? | Насколько от средней цены по истории можно покупать валюту. Работает как защита от покупки валюты на "хаях". 0.2 - будет покупать только если текущая цена продажи составляет не выше 20% от средней * 2, 0.5 - будет покупать только если текущая цена ниже средней, 1 - будет покупать почти всегда, грубо говоря если текущая цена ниже, чем две средних. (0.5)  |
| telegram | object? | Настройки для Telegram бота и уведомлений. Подробнее в следующем заголовке. |
| telegram.token | string | Токен от Telegram бота. |
| telegram.chatId | integer | ID Вашего чата или канала. |
| telegram.bot | boolean? | Включить/выключить бота с командами. (`Не реализовано`) |
| comission | float | Комиссия биржи, не трогать. (`0.001`) |
| maxBuy | float | Максимальное количество валюты, которое бот может потратить на все валюты. |
`Тип, помеченный знаком вопроса является опциональным`

## Настройка Telegram
Идём к https://t.me/BotFather и пишем ему `/newbot`, тем самым создаем бота. Полученный токен вставляем в наш конфиг. После создаем канал или чат, или вовсе делаете всё через личные сообщения. Добавляете туда своего нового бота и бота для получения ID чата (https://t.me/@rasad_chatidbot). Получаете ID чата и вводите его в конфиг, после чего запускаете бота.

## Расшифровка статусов валюты
На главном экране бота неактивные валюты помечены буквой-статусом. Вот их расшифровка:
| Статус | Расшифровка |
| - | - |
| M | Доступный баланс ниже, чем минимальная сумма покупки. |
| H | Работает avgLimit. Текущая цена выше средней с учетом конфига валюты. |
| T | Трейд идет вверх (валюта растёт долгое время) либо валюта еще не достигла дна. |
| N | Доступное для покупки количество валюты ниже минимальной. |
